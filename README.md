# LLM Fine-Tuning on Jane Austen Text

## 📌 Overview
This project focuses on fine-tuning a GPT-based language model on text from Jane Austen’s novels using Hugging Face Transformers. The goal is to generate text that mimics the style, tone, and structure of classic literature.

## 🧠 What I Did
- Collected and cleaned text data from Project Gutenberg
- Preprocessed text into training-ready format
- Fine-tuned a pre-trained language model (DistilGPT2)
- Evaluated model performance using loss and sample outputs
- Generated text samples before and after fine-tuning for comparison

## 🛠️ Tech Stack
- Python
- Hugging Face Transformers
- PyTorch
- Google Colab

## 📊 Results
- Improved text coherence after fine-tuning
- Generated stylistically consistent outputs resembling Jane Austen’s writing
- Observed reduction in perplexity and training loss over epochs
